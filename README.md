## Installation

To run this project locally:

1. Clone the repo:
   ```bash
   git clone https://github.com/Sahilsharma7429/restaurant-geritch.git
   ```

2. Navigate into the project directory:
   ```bash
   cd restaurant-geritch
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm start
   ```
